## Wallpaper Licensing

### Current Status: Alpha (Private)

During alpha development, wallpapers may include images from various sources
for testing purposes. These are NOT cleared for public release.

### Before Public Release (Beta/v1.0)

**Action Required:**

- [ ] Remove all dharmx/walls sourced wallpapers (unlicensed collection)
- [ ] Remove any wallpapers without verified licenses
- [ ] Replace with properly licensed alternatives:
  - Unsplash (free, attribution appreciated)
  - Pixabay (free, no attribution required)
  - Pexels (free)
  - Creative Commons (CC0, CC-BY)
  - AI-generated original artwork
  - Self-created/commissioned artwork
- [ ] Create WALLPAPER-SOURCES.md documenting each wallpaper's source/license
- [ ] Add license headers or attribution file per theme

### Approved Wallpaper Sources

| Source | License | Attribution | Commercial |
|--------|---------|-------------|------------|
| Unsplash | Unsplash License | Appreciated | ✅ Yes |
| Pixabay | Pixabay License | Not required | ✅ Yes |
| Pexels | Pexels License | Not required | ✅ Yes |
| CC0 | Public Domain | Not required | ✅ Yes |
| CC-BY | Creative Commons | Required | ✅ Yes |
| AI-generated | Varies by tool | Check terms | Usually yes |

### Prohibited Sources (Without Permission)

- DeviantArt (artist copyright)
- Pixiv (artist copyright)
- Pinterest (aggregated, unknown sources)
- Reddit (varies, often copyrighted)
- dharmx/walls (unlicensed collection)
- Random Google Images
